# Gestamp (gestamp)

Gestamp Automocion is a Spanish global Tier 1 automotive supplier headquartered in Madrid. Gestamp designs and manufactures metal components for vehicle bodies, chassis, and mechanisms, with focus on lightweighting and structural safety for global automakers.
